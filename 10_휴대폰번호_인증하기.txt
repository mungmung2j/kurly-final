휴대폰번호_인증하기.txt

1. 휴대폰번호 입력하면 => 우측 인증번호 받기 버튼 활성화
2. 인증번호 발송 숫자 6자리 랜덤번호 => 카운트 3분 타이머 동작, 그리고 인증번호 받기 버튼 비활성
3. 3분 타이머 경과시 => 
컨펌모달창 메시지 와 그리고 우측 인증번호 받기 버튼 활성화
"유효 시간이 만료되었습니다. \n 줄바꿈
다시 시도해 주세요."
4. 컨펌모달창 확인버튼 클릭하여 다시 인증번호 받기 버튼 클릭한다.
5. 인증번호가 발송되었습니다. => 
가. 인증번호입력상자와 그리고 인증번호 확인버튼(비활성) 보인다.
나. 인증번호발송 숫자 6자리 랜덤번호 그리고 카운트 3분타이머 동작
 
6. 인증번호 1자이상 입력하면 인증번호 확인버튼(활성)
7. 인증번호 확인버튼 클릭 
인증번호 비교 입력번호 맞으면 
    => 인증에 성공 하였습니다. 
    => 타이머 정지 
    => 다른번호 인증 버튼 보인다.
    => 인증번호 입력 상자 내용 지운다.
    => 인증번호 입력 상자 버튼 모두 숨긴다.
인증번호 비교 입력번호 틀리면 
    => 잘못된 인증 코드 입니다. 
    => 타이머는 계속 진행중

8. 다른번호 인증 버튼 클릭 => 
가. 인증번호 받기로 변경(비활성)
나. 입력상자 지우고[상태관리변수 내용지우기] => 휴대폰:''
다. 포커스 입력대기상태로 커서 깜빡깜빡[useRef()사용] 대기한다.

=> 처음으로 돌아감
